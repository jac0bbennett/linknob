Twitter to CSV

  1. Enter query into input.
  2. For tweets that only contain the words next to each other, wrap the query in " " (double quotes)
  3. Click submit.
  4. The server will attempt to get a maximum of 5000 tweets.
  5. Retweets are filtered out and will subquently reduce number of tweets returned.
  6. Note: Several queries in a short period of time may exceed the twitter API rate limit. This will cause the request to be delayed for up to 15 minutes.
